# Agent Beat UI 重设计方案

## 设计原则

1. **颜色克制** — 只用少量强调色，其余用灰阶
2. **信息层级** — 重要信息突出，背景噪音弱化
3. **呼吸感** — 大量留白，不拥挤
4. **一致性** — 两 Tab 共享设计语言

---

## 配色方案

### 语义色（仅用于关键状态）

| 语义 | 亮色模式 | 暗色模式 | 用途 |
|------|----------|----------|------|

## 布局设计

### 全局布局

```
┌─────────────────────────────────────────────────┐
│  🧠 Agent Beat    [调用链] [仪表盘]    🌙  [设置] │  ← 顶部导航栏（固定）
├─────────────────────────────────────────────────┤
│                                                 │
│                   内容区域                        │
│                                                 │
└─────────────────────────────────────────────────┘
```

- 顶部导航栏固定，高度 56px
- Tab 切换用下划线指示器（不用按钮样式）
- 右侧：主题切换 + 设置入口
- 内容区域占满剩余高度，可滚动

---

### Tab 1：调用链（Call Chain）

#### 信息层级

| 层级 | 内容 | 视觉处理 |
|------|------|----------|
| **L1 关键** | 错误、慢调用（>5s）、MCP 调用 | 红色/琥珀色左边框 + 图标标记 |
| **L2 重要** | 工具名称、耗时、输入摘要 | 正常字重，工具类型色标签 |
| **L3 背景** | 时间戳、session ID、父调用关系 | 弱化灰色，小字号 |

#### 布局草图

```
┌─ 会话卡片 ──────────────────────────────────────┐
│  📂 agent-trace (3 次调用 · 12.3s)     14:32    │  ← 会话标题
├─────────────────────────────────────────────────┤
│                                                 │
│  ┌─ 调用行 ──────────────────────────────────┐  │
│  │ 🔵 Bash    git commit -m "feat..."  1.2s  │  │  ← 正常调用（弱化）
│  └───────────────────────────────────────────┘  │
│                                                 │
│  ┌─ 调用行（错误）──────────────────────────┐  │
│  │ 🔴 Bash    npm test                  3.4s  │  │  ← 错误调用（突出）
│  │    ❌ Error: test failed             │  │
│  └───────────────────────────────────────────┘  │
│                                                 │
│  ┌─ 调用行（慢）────────────────────────────┐  │
│  │ 🟡 Read    src/server.js            8.7s  │  │  ← 慢调用（警告色）
│  └───────────────────────────────────────────┘  │
│                                                 │
└─────────────────────────────────────────────────┘
```

#### 调用行设计

```
┌──────────────────────────────────────────────────┐
│ [类型色条] 📌图标  工具名  输入摘要    耗时  ▼展开 │
│              ↳ 子调用缩进显示                      │
└──────────────────────────────────────────────────┘
```

- 左侧 3px 色条：正常=透明，错误=红，慢=琥珀
- 工具名粗体，类型色标签
- 输入摘要截断显示（hover 完整）
- 耗时右对齐，>5s 琥珀色，>10s 红色
- 展开/折叠：显示完整输入输出

#### 交互

- 顶部：项目选择器 + 搜索框 + 工具类型筛选
- 搜索实时过滤（debounce 300ms）
- 点击调用行展开详情
- 滚动加载（虚拟滚动，性能考虑）

#### 卡片设计规范

- 统一圆角 12px，无边框（用 `box-shadow` 区分层级）
- 卡片内 padding 24px
- 数字用大字号（24px+），标签用弱化灰色
- 图表卡片最小高度 200px

---

## 暗色模式

- 所有颜色通过 CSS 变量切换
- `prefers-color-scheme` 自动检测 + 手动覆盖
- 暗色模式下阴影改为更弱的边框（`border: 1px solid #262626`）

---

## 响应式

- 桌面优先（≥1024px）
- 平板适配（768-1024px）：卡片两列
- 手机不专门适配（开发者工具，主要在桌面用）
